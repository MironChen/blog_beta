更新日志 1

版本号：V1.1

  为什么叫 V1.1 呢？因为之前这个博客存在过 V1.0，只可惜后来因为在将存放博客文件夹的电脑上更新到 macOS High Sierra 的时候发生了 出现“Disk Password”的 Bug，于是电脑便被锁死，只能抹盘，因此博客 V1.0 也就跟随抹去的数据一起消失了。

  本地文件消失的时候，部署在 GitHub 的服务端文件是没有消失的，只可惜，云端上只有 HTML 文件。于是，我就只能重制，尽管有过经验，但第一次重制还是出现了许许多多的问题，网页也始终不能远程访问，重制计划便搁置了下来。无独有偶，突然又搬回计划，这一次竟然用了不到20分钟就完成了，添加了许多功能后，mironchen.me，再一次上线了。

特点

  • 简洁、优雅的个人博客（使用 Hexo 及 Casper 主题搭建）

计划更新

  • 相册功能
  • 点击图片放大显示
  • 将服务端迁移至 DigitalOcean 服务器以优化访问速度
  • 使用 SSL 加密以确保访问安全
